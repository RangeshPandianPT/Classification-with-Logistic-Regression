
Logistic Regression - Breast Cancer Classification

Files included:
- breast_cancer_cleaned.csv: Cleaned dataset
- logistic_regression_model.py: Python code
- logistic_regression_model.ipynb: Jupyter Notebook version
- roc_auc_curve.png: ROC-AUC Curve image
- README.txt: Project summary

This project performs binary classification using logistic regression on breast cancer data,
evaluates the model using various metrics, and includes ROC-AUC visualization and threshold tuning.
